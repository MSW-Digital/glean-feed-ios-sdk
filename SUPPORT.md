# Support

The Glean Feed iOS SDK is in **beta**. For help:

- **Docs** — [gleanfeed.com/docs/ios-sdk](https://gleanfeed.com/docs/ios-sdk/overview):
  install, identity, notifications, diagnostics, and troubleshooting.
- **Your identifiers** — the dashboard, under **Settings → Connections → iOS SDK**
  (workspace ID, slug, and a prefilled setup snippet).
- **Bugs & feedback** — open an issue with the **Bug report** or **Beta feedback**
  template.

Never paste tokens, user emails, or your workspace **widget secret** into a public
issue. Signatures are generated on your backend — the secret never belongs in the
app or in a bug report.

Beta caveat: the public API may change before `1.0.0`. Pin an exact version and
read the [changelog](CHANGELOG.md) before upgrading.
